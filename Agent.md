moonyacc是一个设计类似OCaml Menhir的Parser Generator, 其Parser生成文件后缀为`.mbty`.

`.mbty`文件的基本格式为:

```

一些基本设置

很多个token定义

%%

很多个规则定义

```

moonyacc规则大致可分为内联/非内联两种

+ 内联规则以`%inline`为开始，之后的语法与非内联规则相同
+ 非内联规则以规则名为开始

两者都以分号结束。

最普通的非内联规则语法:

```
规则名 '->' 规则返回值类型
  ':' 匹配模式1 '{' MoonBit表达式 '}'
  | 匹配模式2 '{' ... '}'
  ...
  ;
```

每个匹配模式对应的MoonBit表达式类型应该与标注的规则返回值类型一致。
